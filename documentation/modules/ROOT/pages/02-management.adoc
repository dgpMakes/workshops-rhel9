== RHEL Management with Cockpit

RHEL includes a graphical tool for management. It is named **Cockpit**.

This tool is modular so you will have to install **Cockpit** and after that you will need to install the modules for the admin tasks you want to perform:

* Users management
* Software management
* Storage management
* Virtual machines management
* Session recording
* ...
